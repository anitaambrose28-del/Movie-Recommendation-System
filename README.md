# Movie Recommendation System

A content-based movie recommendation system built with Python and Scikit-learn.

## Features
- Recommends similar movies based on a movie title
- Uses TF-IDF vectorization
- Calculates similarity using cosine similarity
- Handles misspelled movie names using difflib

## Technologies
- Python
- Pandas
- NumPy
- Scikit-learn

## Dataset
The project uses the `movies.csv` dataset.

## How to Run

1. Install the required packages

```bash
pip install -r requirements.txt
```

2. Open the notebook

```bash
jupyter notebook movie_recommendation.ipynb
```

3. Run all cells and enter a movie title.
